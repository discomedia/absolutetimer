# App Store Assets

This directory contains App Store Connect launch material for Absolute Timer.

- `submission.md`: Product page copy, privacy answers, review notes, and asset paths.
- `pages/privacy.html`: Static privacy policy page draft.
- `pages/support.html`: Static support page draft.
- `screenshots/en-US/`: iPhone 6.9-inch portrait screenshots.
- `tools/generate_screenshots.swift`: Local screenshot generator for the PNG assets.

The privacy policy and support page are written for:

- `https://discomedia.co/privacy`
- `https://discomedia.co/support`
